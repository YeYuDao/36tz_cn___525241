<br/></br>### 课程介绍:
Tornado是一种Web服务器软件的开源版本。Tornado和现在的主流 Web 服务器框架(包括大多数Python的框架)有着明显的区别：它是非阻塞式服务器，而且速度相当快。得利于其 非阻塞的方式和对epoll的运用，Tornado每秒可以处理数以千计的连接，因此Tornado是实时Web服务的一个理想框架。
